# DetainShortage
protect or save shortage by translating


### 模块

```
> * web_app         移动端模块
> * web_official    官网模块
> * web_routine     小程序模块
> * web_server      服务器
```

### web_server      服务器
```
> * 引用koa + graphQL
> * 开发环境需要安装babel相关插件
> * 开发环境建议安装nodemon，不用一直开关服务

```

### 公共文件处理方式
```
> * 采用cdn方式存储文件（注意cdn缓存较大）
